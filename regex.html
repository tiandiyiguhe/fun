Common Regex Cheat Sheet

          

          

            

              

                
Classes

                

                  
.
Any char except newline

                  
\d
Digit

                  
\w
Word char

                  
\s
Whitespace

                  
[abc]
a, b, or c

                  
[^abc]
Not a, b, or c

                

              

              

                
Quantifiers

                

                  
*
0 or more

                  
+
1 or more

                  
?
0 or 1

                  
{3}
Exactly 3

                  
{2,5}
Between 2 and 5

                  
*?
Lazy version

                

              

              

                
Groups

                

                  
(abc)
Capture

                  
(?:abc)
No capture

                  
(?<x>abc)
Named capture

                  
\1
Backreference

                  
a|b
Either a or b

                

              

              

                
Anchors

                

                  
^
Start of input or line

                  
$
End of input or line

                  
\b
Word boundary

                  
\B
Not boundary

                

              

              

                
Lookaround

                

                  
(?=x)
Before x

                  
(?!x)
Not before x

                  
(?<=x)
After x

                  
(?<!x)
Not after x

                

              

              

                
Flags

                

                  
g
All matches

                  
i
Ignore case

                  
m
^ and $ per line

                  
s
Dot includes newline
